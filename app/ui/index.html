  1. 先填 API Key。

推荐第一句

我想写一个适合女生看的现代悬疑爱情故事，但我不会搭人物和大纲，请一步一步带我开始。

推荐第二句

请不要一次问我太多问题。你直接给我 3 个方向，并替我选最稳的那个继续做下去。

推荐第三句

先不要写正文，先把人物关系、冲突和章节大纲做清楚。

# Code of Conduct

## Our Standard

We want Story Forge Pro to be welcoming to beginners, writers, builders, and open-source contributors from different backgrounds.

## Expected Behavior

- be respectful
- be constructive
- assume good intent
- explain clearly
- help newcomers without gatekeeping

## Unacceptable Behavior

- harassment or insults
- discriminatory language
- intimidation or threats
- publishing private information without permission
- hostile dismissal of beginner questions

## Enforcement

Project maintainers may remove comments, issues, or contributions that violate this code of conduct.

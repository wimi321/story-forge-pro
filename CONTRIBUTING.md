# Contributing

Thanks for helping improve Story Forge Pro.

## What We Welcome

- better beginner onboarding
- stronger novel and screenplay templates
- clearer docs and translations
- reliability fixes for launch and setup
- workflow improvements for long-form writing

## Development Flow

1. Fork the repo.
2. Create a branch from `main`.
3. Make focused changes.
4. Run:

```bash
npm install
npm run check
```

5. Open a pull request with a clear test note.

## Contribution Standards

- Keep the project beginner-friendly.
- Prefer explicit workflows over vague AI magic.
- Do not add legal or licensing risk.
- Do not commit secrets, tokens, or personal data.
- Keep docs readable for non-technical writers.

## Pull Request Checklist

- The feature solves a real user problem.
- Docs were updated if behavior changed.
- New assets and copy are consistent with the project voice.
- `npm run check` passes.

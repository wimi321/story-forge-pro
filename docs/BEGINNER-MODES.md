# Beginner Modes

Story Forge Pro now follows a very simple principle:

- ask less
- decide more
- output something usable every time

## Beginner-first defaults

- default language: Simplified Chinese
- default writing behavior: guided mode
- default interaction: options instead of theory
- default output: outline, cards, scenes, or draft-ready materials

## Best entry points

### Absolute beginner

```bash
npm start
```

### I already know I want a novel

```bash
npm run novel
```

### I already know I want a screenplay

```bash
npm run screenplay
```

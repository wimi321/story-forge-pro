# Beginner Guide

## If You Have Never Used AI for Writing Before

Start here.

### Best first command

```bash
npm run novel
```

Then say something simple like:

```text
我想写一个适合女生看的现代悬疑爱情故事，但我不会搭人物和大纲，请带我一步一步开始。
```

## Best Practice

- start with audience before plot
- start with conflict before worldbuilding
- start with outline before drafting long chapters
- ask for options, not perfection
- revise in passes: structure, then scene, then line-level polish

## Recommended Session Order

1. concept and positioning
2. lead character and opposition
3. outline and chapter map
4. draft chapter 1
5. revise hooks and pacing

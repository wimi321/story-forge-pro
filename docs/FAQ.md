# FAQ

## Is this an official Anthropic project?

No. Story Forge Pro is an independent wrapper layer.

## Do I need to know prompt engineering?

No. The project is designed to guide beginners.

## Can I use it for Chinese web novels?

Yes. The Chinese prompt stack and workflows are optimized for that use case.

## Can I use it for screenplays?

Yes. There is a dedicated screenplay mode.

## What if Claude Code changes upstream?

The launcher supports `CLAUDE_CODE_CLI_PATH` as an escape hatch for custom environments.
